Conway's Game of Life
=====================

Use
***

Run :bash:`make` to compile. You'll need to have the ncurses library installed.
Run :bash:`./life` to execute.

- 'p' to play/pause
- 'r' to fill randomly (r = .5)
- 'f' to draw an f-pentamino
- 'g' to draw a glider
- 'q' to quit

.. role:: bash(code)
   :language: bash
